Featured Documentation

The documentation is intended to help you properly install and configure the theme. I am happy to help with any questions, so feel free to leave a comment on my profile if you need help.

Installation

  1. Download the theme archive.
  2. Navigate to ghost/content/themes.
  3. Paste the theme archive and unzip it.
  4. Restart Ghost and log in to your Ghost dashboard at yourdomain.com/ghost/
  5. Navigate to Settings » Themes and select featured from the dropdown menu.
  6. Click Save.

Configuration

Configuring Featured is extremely easy and intuitive.

Add Logo/Avatar

  1. Go to Settings » General.
  2. Upload your image in the Blog Logo section. The image you upload here will be automatically added as logo/avatar in the blog header.

Add Name & About/Bio

  1. Go to Settings » General.
  2. Type your name in the Blog Title input field. The name you enter here will be automatically used in the theme header as your name.
  3. Type your preferred blog description in the Blog Description input field. The text you enter here will be automatically used as your blog info in the theme header.

Add Social Media

  1. Navigate to the featured/partials/social.hbs file.
  2. Replace the "#" values with your profile links for Facebook, Twitter, Google+, and Dribbble.
  3. If you want to remove any social icon from the theme, simply delete for example <a href="#" target="_blank" class="icon-googleplus"></a> from the file.

Set Up "Hire Me" Section

  1. Navigate to the featured/partials/hireme.hbs file.
  2. Replace the Like what I do? I can do even more awesome stuff for you! text with anything you prefer.
  3. Right below this text is the Hire Me button. When the user clicks on this button, he can send you an email. To receive the email, replace youremail@ghost-featured.com with your email address.

Important About Posts

  • In order for the thumbnails to show on the index of your blog properly, it is highly recommended to start each post with an image. Your first image from the post will be used as the thumbnail. The recommended image orientation for the thumbnails is a landscape format.
  • Posts support video, however the current theme version does not support video as thumbnails. It will be available in the coming Featured version.
